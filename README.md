# 📖 Theory

Conditional statements are used in Python to make decisions based on conditions. They allow a program to execute different blocks of code depending on whether a condition is True or False.

Python uses the following conditional statements:

if statement – executes a block of code if the condition is true

if-else statement – executes one block if the condition is true, otherwise executes another block

if-elif-else statement – checks multiple conditions and executes the block for the first true condition

nested if statement – an if statement inside another if statement

Conditional statements mainly use comparison operators like:

==, !=, >, <, >=, <=

and logical operators like:

and, or, not
